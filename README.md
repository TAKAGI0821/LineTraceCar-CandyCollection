- **工夫点**:
アッカーマン機構というモノを採用することで、左タイヤと右タイヤの水平位置のずれを最小限に抑えている。
また、これにより、タイヤによる前進方向の力の妨害を防ぐことに寄与している。<br>
左側：は両輪を1つの軸でつないでいる。<br>
右側：アッカーマン機構でタイヤをつけている。<br>
